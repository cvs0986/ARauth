## Setup Steps

### Step 1: Install GitHub CLI

```bash
sudo dnf install gh
```

### Step 2: Authenticate

```bash
gh auth login
```

Follow the prompts:
- Choose GitHub.com
- Choose HTTPS or SSH
- Authenticate via browser or token

### Step 3: Verify

```bash
gh auth status
```
